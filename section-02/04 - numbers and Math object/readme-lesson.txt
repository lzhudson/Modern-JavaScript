Objeto Math 
É um objeto que contem alguns metodos e atributos matematicos para que possamos usar.
Math.PI; // Retorna o valor de PI.
Math.E; // Retorna a base do logaritimo.
Math.round(value); // Arredonda o numero para o valor inteiro mais proximo.
Math.ceil(value); // Arredonda o numero para o valor inteiro maior mais proximo.
Math.floor(value); // Arredonda o numero para o valor inteiro menor mais proximo.
Math.srqt(value); // Retorna o valor da raiz do numero passado.
Math.abs(value); // Retorna um valor inteiro positivo.
Math.pow(num, elevado); // Retorna o valor da potenciação.
Math.min(arguments); // Retorna o menor numero passado por parametro.
Math.max(arguments); // Retorna o maior numero passado por parametro.
Math.random(); // Retorna um numero aleatorio.
Math.floor(Math.random() * 20 + 1); // Retorna um numero inteiro de 1 a 20;
